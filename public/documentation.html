Movie API

GET /movies

Returns a JSON array containing all movies.

GET /movies/{title}

Returns a JSON object representing a movie with the specified title.

Movie Object:

  • Title: The title of the movie.
  • Description: A brief description of the movie.
  • Genre: The genre of the movie.
  • Director: The name of the director.

GET /movies/genre/{genreName}

Returns a JSON object representing the genre with the specified name.

Genre Object:

  • Name: The name of the genre.
  • Description: A brief description of the genre.

GET /movies/directors/{directorName}

Returns a JSON object representing the director with the specified name.

Director Object:

  • Name: The name of the director.
  • Bio: A brief biography of the director.

GET /users

Returns a JSON array containing all users.

GET /users/{id}

Returns a JSON object representing the user with the specified ID.

User Object:

  • ID: The unique identifier of the user.
  • Username: The username of the user.
  • Email: The email address of the user.
  • Password: The password of the user.
  • Birthday: The birthday of the user.
  • FavoritesMovies: An array of favorite movies added by the user.

POST /users

Allows new users to register. Expects a JSON object containing user details in the request body. Returns the created user object.

PUT /users/{id}

Allows users to update their username. Expects a JSON object containing the updated user details in the request body. Returns the updated user object.

POST /users/{id}/{movieTitle}

Allows users to add a movie to their favorites list. Returns a confirmation message.

DELETE /users/{id}/{movieTitle}

Allows users to remove a movie from their favorites list. Returns a confirmation message.

GET /

Returns a simple text response.

Static Files

Serves static files located in the "public" folder.

Error Handling

Logs errors to the console and sends a 500 response.
