Movie API

2.5 Rest points

Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET None A JSON object holding data about a single movie, containing title, year, genre, director. Example: { 'Genre': { 'Name': 'Action', 'Description': 'Action film involving intense conflicts, often with a central superhero or protagonist.' }, 'Director': { 'Name': 'Christopher Nolan', 'Bio': 'Best known for his cerebral, often nonlinear, storytelling, acclaimed writer-director Christopher Nolan has made a significant impact in the world of filmmaking.', 'Birth': 'July 30, 1970' }, 'Title': 'The Dark Knight', 'Description': 'In this superhero film, Batman faces the Joker, a criminal mastermind who seeks to create chaos in Gotham City. The battle between the two forces Batman to confront his own moral limits.', }
Return data about a genre by name /movies/genre/[genreName] GET None A JSON object holding data about a single genre, containing genre name, description. Example: { 'Name': 'Action', 'Description': 'Action film involving intense conflicts, often with a central superhero or protagonist.' },
Return data about a director by name /movies/directors/[directorName] GET None A JSON object holding data about a single director, containing director name, bio, birth and death year. Example: { 'Name': 'Christopher Nolan', 'Bio': 'Best known for his cerebral, often nonlinear, storytelling, acclaimed writer-director Christopher Nolan has made a significant impact in the world of filmmaking.', 'Birth': 'July 30, 1970' }
